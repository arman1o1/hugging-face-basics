# Ollama Setup for Jupyter

## 1. Install Ollama

Download and install from: **[ollama.ai](https://ollama.ai)**

Or via command line:

- **Windows**: `winget install Ollama.Ollama`
- **Mac**: `brew install ollama`
- **Linux**: `curl -fsSL https://ollama.com/install.sh | sh`

## 2. Quick Start

Run these commands in a **Jupyter terminal cell**:

```bash
# Pull a model (one-time)
!ollama pull llama3.2

# Start the server (run in background)
!ollama serve &
```

## Alternative: System Terminal

If the above doesn't work, open a separate terminal:

```bash
ollama serve
```

Keep it running while using the notebook.

## Verify Connection

```python
from langchain_ollama import ChatOllama
llm = ChatOllama(model="llama3.2")
```
